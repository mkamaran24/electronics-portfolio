# ElektraComponents - Electronic Components Supplier

This is a one-page portfolio website for an electronic components trading company based in Erbil, Iraq.

## Tech Stack

- **Framework**: Next.js 15 (App Router)
- **Styling**: Tailwind CSS 4
- **Animations**: Framer Motion
- **Icons**: Lucide React
- **Language**: TypeScript

## Getting Started

1. Install dependencies:
   ```bash
   npm install
   ```

2. Run the development server:
   ```bash
   npm run dev
   ```

3. Open [http://localhost:3000](http://localhost:3000) with your browser.

## Project Structure

- `src/app/page.tsx`: Main entry point assembling all sections.
- `src/components/sections/`: Individual page sections (Hero, About, Products, etc.).
- `src/components/layout/`: Global layout components (Navbar).
- `src/lib/utils.ts`: Utility functions (cn).

## Deployment

This project is optimized for deployment on Vercel.
